# KU Lost & Found
## 이 프로그램은 고려대학교 학생들을 위한 통합 분실물 관리 프로그램입니다.
### 환영합니다!

**고려대학교** 캠퍼스 내 분실물 및 습득물을 쉽게 등록하고 찾을 수 있는 웹 애플리케이션입니다.

- 분실물 또는 습득물 정보를 등록할 수 있습니다.
- 검색어를 통해 원하는 물품을 빠르게 찾을 수 있습니다.
- 상태별 및 유형별 필터링이 가능합니다.

| 기능 | 설명 |
|------|------|
| 홈 화면 | 최근 등록된 분실물/습득물 확인 |
| 검색 기능 | 물건명, 장소 등으로 검색 |
| 필터 기능 | 분실물/습득물 및 상태별 조회 |
| 회원 기능 | 로그인 및 회원가입 |

- 사용 예시는 다음과 같습니다.

> 중앙도서관 1층에서 아이패드를 분실한 학생이 게시글을 등록합니다.
> 다른 학생이 습득물 목록에서 해당 게시글을 확인한 후 제보하여 분실물을 찾을 수 있습니다.
